## Prerequisite

- Git
- Node.js (v22.16.0 and above)
- npm

## Clone project using CMD (Windows) or Terminal (Linux/MacOS)

```bash
$ git clone https://github.com/thampaponn/typescript-merge.git

$ cd typescript-merge
```

## Project setup

```bash
$ npm install
```

## Start service

```bash
$ npm run start

```
**See result in the terminal**

## Run unit tests

```bash
$ npm run test
```
